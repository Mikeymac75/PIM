Instructions for Excel File Format:

Please ensure your Excel file (.xlsx) has the following columns in the first sheet. The order of columns matters, and the first row should be headers exactly matching these names (e.g., name, associated_product_name, etc.).

  • name (Required): The specific name for this production batch (e.g., "Tomato Batch Spring 2024", "Basil Pot 3"). Text format.
  • associated_product_name (Optional): If this production item yields a specific product already defined in your 'Products' list (e.g., "Tomatoes", "Basil Leaves"), enter its name here. If the product name is provided but not found, the association will be skipped. Text format.
  • plant_date (Required): The date the item was planted or started. Format: YYYY-MM-DD.
  • time_to_harvest_days (Required): Estimated number of days from planting until the first harvest can occur. Integer (e.g., 60).
  • expected_harvest_period_days (Required): Estimated number of days over which harvesting can occur. Integer (e.g., 30).
  • expected_yield_total (Required): The total expected yield from this production item over its entire harvest period, in the unit of the associated product (if any) or a general unit if no product is associated. Number (e.g., 5.5).
  • status (Optional): The current status of the item. Must be one of "Growing", "Harvesting", or "Finished". If left blank, it will default to "Growing". Text format.

Important:

  • Data validation is performed. Rows with missing required fields or incorrect data types will be skipped.
  • If `associated_product_name` is given, it must match an existing product name in your system for the link to be made.
