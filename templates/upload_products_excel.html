Instructions for Excel File Format

Please ensure your Excel file (.xlsx) has a header row with the following column names (case-insensitive, order does not matter):

  • name (Required): Name of the product.
  • category_name (Required): Name of an existing category.
  • subcategory_name (Optional): Name of an existing subcategory. If provided, category_name must also be specified, and the subcategory must belong to that category.
  • unit_of_measure (Required): e.g., kg, pcs, liter.
  • default_expiry_days (Required): Number of days until expiry from purchase date (e.g., 7, 14).
  • par_level (Optional): Desired minimum stock level (float, defaults to 0 if blank).
  • max_holding_amount (Optional): Maximum stock level to hold (float, defaults to 0 if blank).
  • purchase_location (Optional): e.g., Sobeys, Costco.

The first sheet in the Excel file will be processed. Rows after the header row will be treated as product data.

"Duplicates" are identified by product name. "Overwrite" will update all fields of the existing product with the data from the Excel row.
