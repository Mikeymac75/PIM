Instructions for Excel File Format:

Please ensure your Excel file (.xlsx format) has the following columns in the first sheet:

  • Name: The name of the grocery item (Text).
  • Quantity: The quantity of the item (Text or Number, e.g., "10 bags", "2", "1.5 liters").
  • Purchase Date: The date of purchase in YYYY-MM-DD format (e.g., 2023-10-28).
  • Expiry Days: The number of days from the purchase date until the item expires (Number, e.g., 7, 14, 30).

The first row of the sheet should be the header row containing these exact column names. The order of columns does not matter, but the names must match.

Example:

Name Quantity Purchase Date Expiry Days
Apples 10 bags 2023-10-01 14
Milk 2 gallon 2023-10-15 7
