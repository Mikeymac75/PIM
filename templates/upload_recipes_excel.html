Instructions for Excel File Format (Recipes):

Please ensure your Excel file (.xlsx format) has the following columns in the first sheet. The first row should be the header.

  • Recipe Name: (Required) The name of the recipe.
  • Description: (Optional) A brief description of the recipe.
  • Instructions: (Optional) Detailed cooking instructions.
  • Output Product Name: (Optional) If the recipe produces a specific inventory item, enter its name here. The product must already exist in the system.
  • Output Yield: (Optional, but Required if 'Output Product Name' is provided) The quantity of the output product produced by one batch of the recipe.
  • Ingredient 1 Name: (Required if any ingredient in this numbered set is provided) Name of the first ingredient. Must match an existing product name in the inventory system.
  • Ingredient 1 Quantity: (Required if 'Ingredient 1 Name' is provided) Quantity of the first ingredient (e.g., "2", "0.5", "100g").
  • Ingredient 1 Notes: (Optional) Any notes for the first ingredient (e.g., "chopped", "unsalted").
  • Ingredient 2 Name: (Optional) Name of the second ingredient.
  • Ingredient 2 Quantity: (Required if 'Ingredient 2 Name' is provided) Quantity of the second ingredient.
  • Ingredient 2 Notes: (Optional) Notes for the second ingredient.
  • ... (Continue this pattern for more ingredients, e.g., up to Ingredient 15 Name, Ingredient 15 Quantity, Ingredient 15 Notes)

Each row after the header represents a single recipe. If a recipe has fewer ingredients than the maximum number of ingredient columns, leave the subsequent ingredient columns empty for that recipe's row.
