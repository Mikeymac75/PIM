Instructions for Excel File Format

Please ensure your Excel file (.xlsx) has a header row with the following column names (case-insensitive, order does not matter). The first sheet in the Excel file will be processed.

Expected Columns:
  • product_name (Required): Name of the product. Must match an existing product in the system.
  • quantity_consumed_this_time (Required): The amount of the product consumed in this historical entry. Must be a positive number.
  • consumed_date (Required): The date the consumption occurred, in YYYY-MM-DD format.
  • original_quantity_string (Optional): The original quantity string of the batch from which this item was consumed (e.g., "1 kg", "12 units"). Text format.
  • purchase_date (Optional): The purchase date of the batch from which this item was consumed, in YYYY-MM-DD format.
  • expiry_date (Optional): The expiry date of the batch from which this item was consumed, in YYYY-MM-DD format.

Note: This upload process appends new historical records. It does not update or delete existing historical entries.
